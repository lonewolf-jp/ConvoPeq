## 6. CallbackTelemetryScope（B: processTime 取得）

**ファイル**: `src/audioengine/AudioEngine.Processing.AudioBlock.cpp`（約62-93行目）

現在:
```cpp
struct CallbackTelemetryScope final {
    // ...
    CallbackTelemetryScope(...) noexcept
        : startUs(enabled ? convo::getCurrentTimeUs() : 0) {}  // ★ enabled時のみ
    ~CallbackTelemetryScope() noexcept {
        if (enabled) { /* ... */ }
    }
};
```

改修後:
```cpp
struct CallbackTelemetryScope final {
    AudioEngine& engine;
    int samples;
    uint64_t startUs;
    uint64_t callbackIndex;
    uint64_t expectedIntervalUs;

    CallbackTelemetryScope(AudioEngine& owner, int numSamplesIn,
                           uint64_t cbIdx, uint64_t expectedUs) noexcept
        : engine(owner), samples(numSamplesIn)
        , startUs(convo::getCurrentTimeUs())        // ★ 常時取得
        , callbackIndex(cbIdx)
        , expectedIntervalUs(expectedUs) {}

    ~CallbackTelemetryScope() noexcept {
        const uint64_t endUs = convo::getCurrentTimeUs();
        const uint64_t processTime = (endUs > startUs) ? (endUs - startUs) : 0;
        if (owner.isCliProcessingTelemetryEnabled())
            engine.recordAudioCallbackProcessingStats(samples, processTime);
        // ★ B: リングバッファ書込（常時）
        const uint64_t wc = convo::fetchAddAtomic(
            engine.rtLocalState_.callbackTimingWriteCount,
            uint64_t{1}, std::memory_order_relaxed);
        const size_t idx = (wc - 1) % decltype(engine.rtLocalState_)::kCallbackTimingSlots;
        auto& entry = engine.rtLocalState_.callbackTimingHistory[idx];
        entry.callbackIndex = callbackIndex;
        entry.processTimeUs = processTime;
        entry.driftUs = engine.rtLocalState_.lastCallbackDriftUs.load(relaxed);
        entry.cpu = engine.rtLocalState_.lastCallbackProcessor.load(relaxed);
        const double budgetPct = (expectedIntervalUs > 0)
            ? (static_cast<double>(processTime) / expectedIntervalUs) * 100.0 : 0.0;
        entry.budgetPermille = static_cast<uint16_t>(
            std::min(999.0, std::round(budgetPct * 10.0)));
        entry.expectedIntervalUs = static_cast<uint32_t>(expectedIntervalUs);
        entry.sequence.store(callbackIndex, std::memory_order_release);
    }
};
```

## 7. GetCurrentProcessorNumber() 可用性（G）

**結果**: `<JuceHeader.h>` → `JUCE` → `<windows.h>` のチェーンで `GetCurrentProcessorNumber()` が利用可能。

`<windows.h>` は以下のパスでインクルード済み:
- `AudioBlock.cpp`: `#include <JuceHeader.h>` → JUCE内部で `<windows.h>`
- `BlockDouble.cpp`: 同上
- `ConvolverProcessor.Runtime.cpp`: 同上

追加の `#include` は不要。
