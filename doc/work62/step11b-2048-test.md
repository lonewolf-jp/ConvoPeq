# Step 11b: バッファサイズ 2048 試験

## 位置づけ
- **原因特定ではなく現象緩和**
- コード変更不要（ASIOドライバ設定のみ）

## 手順
1. Voicemeeter BANANA の設定を開く
2. ASIO バッファサイズを 1024 → **2048** に変更
3. ConvoPeq を起動し XRUN 発生頻度を確認
4. 必要に応じて [DIAG_DRAIN] ログと [CB_ARRIVAL] ログで状態確認

## 効果予測
- 192kHz/2048samples → callback 間隔 10.67ms（現在の2倍）
- 8〜10ms の dispatch 遅延が予算内に収まる可能性あり
- ただしレイテンシは倍増するため、運用用途に応じて判断
