## Step 6: 非同期 Logger（asyncSink）⬜ 未着手

**ファイル**: `src/audioengine/AudioEngine.Timer.cpp`

### LogEntry

```cpp
struct alignas(64) LogEntry {
    uint16_t length;   // 実UTF-8バイト数（null除く）
    char text[254];    // 最大254文字
};
```

### asyncSink

```cpp
static void asyncSink(const juce::String& message) {
    const bool pushed = s_logBuffer.pushWithWriter([&](LogEntry& entry) {
        const size_t copied = message.copyToUTF8(entry.text, sizeof(entry.text));
        entry.length = (copied > 0) ? static_cast<uint16_t>(copied - 1) : 0;
    });
    if (!pushed) s_droppedLogs.fetch_add(1, std::memory_order_relaxed);
}
```

### flushLogBuffer

```cpp
void AudioEngine::flushLogBuffer() {
    std::string batch; batch.reserve(32768);
    LogEntry entry; int count = 0;
    while (s_logBuffer.pop(entry)) {
        batch.append(entry.text, entry.length); batch += '\n';
        if (++count >= 100) {
            juce::Logger::writeToLog(juce::String(batch));
            batch.clear(); count = 0;
        }
    }
    if (count > 0) juce::Logger::writeToLog(juce::String(batch));
    uint64_t dropped = s_droppedLogs.exchange(0, std::memory_order_acq_rel);
    if (dropped > 0) DBG("[LOG_DROP] async log dropped " + ...);
}
```

### フラッシュ呼び出し

別スレッドまたは別Timerから 500ms 周期で `flushLogBuffer()` を呼ぶ。
