#### R17 最小 runbook（Split 移行 / Rollback）

- 前提確認: `retire_timeline.json` の `epochMode` が `shared` であること、`shutdown_trace.json` の `verified=true` を確認する。

- 切替: non-RT 側で `RetireRuntimeEx::setEpochMode(EpochMode::Split)` を呼ぶ。

- 観測: 次回出力された `retire_timeline.json` の `epochMode == "split"` を確認する。

- Rollback 条件: HB/retire 検証が fail した場合、または運用閾値逸脱時。

- Rollback 操作: non-RT 側で `RetireRuntimeEx::setEpochMode(EpochMode::Shared)` を呼び戻し、`retire_timeline.json` で `epochMode == "shared"` を再確認する。
