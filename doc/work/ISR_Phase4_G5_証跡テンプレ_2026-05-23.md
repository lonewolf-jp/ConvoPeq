## 6. 連続性評価

### 連続性評価の埋め方

- 評価対象は `mix=0 -> mix>0` の再mix区間に限定する。
- クリックピークは、再mix直前直後の短区間を見て単発スパイクの有無を記入する。
- 20ms 窓 RMS差は、再mix点を跨いだ区間で測り、レベル段差の大きさを記入する。
- snapshot 遷移影響と bypass 遷移影響は、回帰の有無を簡潔に書く。
- 判定基準を満たした場合のみ `Pass` とし、どれか1つでも外れたら `Fail` とする。
- メモ欄には、クリックと RMS のどちらが主因かを一言で残す。

### 連続性評価の未計測メモ

- 連続性評価は、再mix の比較波形を用意してから埋める。
- PR 添付前の段階では、空欄をそのまま残してもよい。

### 評価対象

- 比較対象: Candidate のみ
