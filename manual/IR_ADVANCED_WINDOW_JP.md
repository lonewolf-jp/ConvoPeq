# IR Advanced Settings – 詳細説明

本ウィンドウでは、インパルス応答（IR）の読み込み後の処理に関する詳細設定を行います。
以下、各コントロールの目的、デフォルト値、設定範囲、および推奨される使い方を説明します。

> **開き方**：メインウィンドウの「IR Advanced...」ボタンをクリックします。

---

## 1. IR Length（IR 長）
- **目的**：コンボリューションに使用する IR の実効長（秒）を指定します。自動検出された長さを上書きし、手動で長さを制限できます。
- **デフォルト**：`IR_LENGTH_DEFAULT_SEC = 1.0` 秒（Auto 時は自動検出値）
- **範囲**：`IR_LENGTH_MIN_SEC = 0.5` ～ `IR_LENGTH_MAX_SEC = 3.0` 秒（0.1 秒刻み）
- **推奨**：
  - 室内補正 IR には 0.5～1.0 秒が適しています。
  - 残響付加（リバーブ）用途では 1.0～3.0 秒が一般的です。
  - 短くすると低域の位相回転やプリリンギングを軽減できますが、残響感も減少します。

---

## 2. Rebuild Debounce（再構築デバウンス）
- **目的**：IR の再構築（リビルド）を遅延させる時間（ms）を設定します。連続的なパラメータ変更をまとめて、負荷の高い再構築の頻度を減らします。
- **デフォルト**：`REBUILD_DEBOUNCE_DEFAULT_MS = 20` ms
- **範囲**：`REBUILD_DEBOUNCE_MIN_MS = 10` ～ `REBUILD_DEBOUNCE_MAX_MS = 3000` ms（10 刻み）
- **推奨**：
  - デフォルト値でほとんどのケースで十分です。
  - 操作中の応答性を高めたい場合は短め（10～20 ms）に、負荷を極力抑えたい場合は長め（100～300 ms 以上）に設定してください。

---

## 3. Mixed Phase Parameters（ミックスドフェーズパラメータ）

Mixed Phase（混合位相）は、インパルス応答（IR）の位相特性を、低域では線形位相（Linear Phase）に近く、高域では最小位相（Minimum Phase）に近づけるように遷移させる処理です。

位相モードはメインウィンドウのコンボボックスで **As-Is / Mixed / Minimum** から選択します。Mixed 選択時のみ以下の f1 / f2 パラメータが有効になります。

- **As-Is（線形位相）**：周波数によって遅延が一定で、波形の歪みは少ない反面、プリリンギング（音の出始め前に発生するリンギング）が発生しやすくなります。
- **Minimum（最小位相）**：エネルギーを時間軸の前方に集中させ、プリリンギングを抑えますが、周波数によって遅延が変化するため位相特性が非線形になります。
- **Mixed（混合位相）**：低域は As-Is の位相特性を維持し、高域は Minimum に徐々に移行します。

### 3.1 Mix Start f（f1）
- **目的**：ミックスドフェーズ遷移の開始周波数（Hz）を設定します。この周波数以下では位相は As-Is 特性が維持されます。
- **デフォルト**：`MIXED_F1_DEFAULT_HZ = 200` Hz
- **範囲**：`MIXED_F1_MIN_HZ = 100` ～ `MIXED_F1_MAX_HZ = 400` Hz
- **推奨**：
  - 低域の位相特性を維持したい場合はデフォルトのまま。
  - より位相補正効果を強めたい場合は低め（100～150 Hz）に設定します。

### 3.2 Mix End f（f2）
- **目的**：ミックスドフェーズ遷移の終了周波数（Hz）を設定します。この周波数以上では位相は Minimum 特性になります。
- **デフォルト**：`MIXED_F2_DEFAULT_HZ = 1000` Hz
- **範囲**：`MIXED_F2_MIN_HZ = 700` ～ `MIXED_F2_MAX_HZ = 1300` Hz
- **推奨**：
  - デフォルト値で自然な位相バランスが得られます。
  - より高い帯域で最小位相特性を適用したい場合は低め（700～900 Hz）に設定します。

---

## 4. Tail Processing（テール処理）

IR の後半部分（テール）の高域特性を調整します。物理的な空気吸収を模擬したり、直接音・初期反射の明瞭さを保持しながらテールのみを処理することができます。

### 4.1 Tail Mode（テールモード）
- **目的**：テール処理の方式を選択します。
- **選択肢**：
  - **Air Absorption**：物理的な空気吸収を模擬し、IR 全体に高域減衰を適用します。減衰は開始周波数以上で強度に応じて連続的に強まります。
  - **Layer Tail Contouring**：L0（先頭部、直接音・初期反射に相当）は無処理とし、L1/L2（後半の残響テール）にのみ減衰を適用します。直接音の明瞭さを保持しつつ、テールのざらつきを低減できます。
  - **Bypass**：テール処理を適用しません。IR をそのまま使用します。
- **デフォルト**：Air Absorption

### 4.2 Tail Start（テール開始位置）
- **目的**：テール処理（減衰）を適用開始する時間位置（秒）を指定します。この位置より前の部分は無処理となります（Layer Tail Contouring では L0 に相当）。
- **デフォルト**：`TAIL_START_DEFAULT_SEC = 0.085` 秒
- **範囲**：`TAIL_START_MIN_SEC = 0.01` ～ `TAIL_START_MAX_SEC = 0.80` 秒（0.005 刻み）
- **推奨**：
  - デフォルト値（85 ms）でほとんどの IR で適切なテール位置が得られます。

### 4.3 Tail Strength（減衰強度）
- **目的**：開始周波数以上の帯域における減衰の強さを指定します。値が大きいほど高域が強く減衰します。
- **デフォルト**：`TAIL_STRENGTH_DEFAULT = 1.0`
- **範囲**：`TAIL_STRENGTH_MIN = 0.0` ～ `TAIL_STRENGTH_MAX = 2.0`（0.01 刻み）
- **推奨**：
  - デフォルトで標準的な減衰効果が得られます。
  - 減衰を強めたい場合は 1.2～1.5、弱めたい場合は 0.5～0.8 に設定してください。

### 4.4 Tail L1-L2 Multiplier（L1/L2 倍率） – Layer Tail Contouring モードのみ
- **目的**：L1/L2 レイヤーに適用される減衰強度の整数倍率を設定します。Layer Tail Contouring モードでは、NUC の L1（中間パーティション）と L2（後半テールパーティション）に個別に減衰が適用され、この倍率で強度が乗算されます。
- **デフォルト**：`TAIL_L1L2_MULT_DEFAULT = 8`
- **範囲**：`TAIL_L1L2_MULT_MIN = 2` ～ `TAIL_L1L2_MULT_MAX = 16`
- **推奨**：
  - デフォルト（8x）でバランスの取れた効果が得られます。
  - テールの減衰をより強めたい場合は大きい値（12～16）に、弱めたい場合は小さい値（2～4）に設定してください。

---

## 5. HCF / LCF / EQ LPF との関係
- **HCF（High-Cut Filter）**、**LCF（Low-Cut Filter）**、**EQ LPF（EQ ローパスフィルター）** は、リアルタイム処理（Audio Thread）で出力信号に適用されるフィルターです。
- テール処理（Air Absorption / Layer Tail Contouring）は IR データに「焼き込まれる」ため、これらとは独立して動作します。
- 全体の高域減衰を調整する場合は、まずテール処理で大まかな特性を決め、必要に応じて HCF や EQ LPF で微調整してください。

---

## 6. リサンプリング位相モード（Resample Phase）

IR のサンプルレートが現在のプロジェクトと異なる場合のリサンプリング処理に適用される位相モードを選択します。

| 項目 | 説明 |
|------|------|
| **Linear (precise)** | 線形位相モード。周波数特性を科学的に正確に保持します。測定・ミキシング用途に適しています。 |
| **Minimum (musical)** | 最小位相モード。アタック成分を保持し自然な聴感が得られます。ギターキャビネット等の音楽用途に適しています。 |

> **注意**：この設定を変更すると IR の再構築（リビルド）がトリガーされます。

---

以上が IR Advanced Settings の全設定項目です。各パラメータは状況に応じて調整し、最適なインパルス応答処理を実現してください。
