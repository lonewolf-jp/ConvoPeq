# IR Advanced Settings – 詳細説明

本ウィンドウでは、インパルス応答（IR）の読み込み後の処理に関する詳細設定を行います。  
以下、各コントロールの目的、デフォルト値、設定範囲、および推奨される使い方を説明します。

---

## 1. IR Length（IR 長）
- **目的**：コンボリューションに使用する IR の実効長（秒）を指定します。自動検出された長さを上書きし、手動で長さを制限できます。
- **デフォルト**：自動検出された長さ（Auto）または 1.0 秒（手動）
- **範囲**：0.5 秒 ～ 最大許容値（サンプルレート依存）
- **推奨**：  
  - 室内補正 IR には 0.5～1.0 秒が適しています。  
  - 残響付加（リバーブ）用途では 1.0～3.0 秒が一般的です。  
  - 短くすると低域の位相回転やプリリンギングを軽減できますが、残響感も減少します。

---

## 2. Rebuild Debounce（再構築デバウンス）
- **目的**：IR の再構築（リビルド）を遅延させる時間（ms）を設定します。連続的なパラメータ変更をまとめて、負荷の高い再構築の頻度を減らします。
- **デフォルト**：400 ms
- **範囲**：50 ～ 3000 ms
- **推奨**：  
  - デフォルト値でほとんどのケースで十分です。  
  - 操作中の応答性を高めたい場合は短め（100～200 ms）に、負荷を極力抑えたい場合は長め（1000 ms 以上）に設定してください。

---

## 3. Mixed Phase Parameters（ミックスドフェーズパラメータ）

Mixed Phase（混合位相）は、インパルス応答（IR）の位相特性を、低域では線形位相（Linear Phase）に近く、高域では最小位相（Minimum Phase）に近づけるように遷移させる処理です。

- **線形位相（As-Is）**：周波数によって遅延が一定で、波形の歪みは少ない反面、プリリンギング（音の出始め前に発生するリンギング）が発生しやすくなります。
- **最小位相（Minimum Phase）**：エネルギーを時間軸の前方に集中させ、プリリンギングを抑えますが、周波数によって遅延が変化するため位相特性が非線形になります。

Mixed Phase は、**低域は線形位相のまま（位相ずれが少なく、定位感を保持）、高域は最小位相に移行（プリリンギングを抑制）** することで、両者の利点をバランスよく取り入れたものです。  
この遷移が始まる周波数と終わる周波数を、以下の `f1`（Mix Start f）と `f2`（Mix End f）で指定します。

### 3.1 Mix Start f（f1）
- **目的**：ミックスドフェーズ遷移の開始周波数（Hz）を設定します。この周波数以下では位相は Linear 特性（最小位相変換前の位相）が維持されます。
- **デフォルト**：200 Hz
- **範囲**：100 ～ 400 Hz
- **推奨**：  
  - 低域の位相特性を維持したい場合はデフォルトのまま。  
  - より位相補正効果を強めたい場合は低め（100～150 Hz）に設定します。

### 3.2 Mix End f（f2）
- **目的**：ミックスドフェーズ遷移の終了周波数（Hz）を設定します。この周波数以上では位相は Minimum 特性（最小位相）になります。
- **デフォルト**：1000 Hz
- **範囲**：700 ～ 1300 Hz
- **推奨**：  
  - デフォルト値で自然な位相バランスが得られます。  
  - より高い帯域で最小位相特性を適用したい場合は低め（700～900 Hz）に設定します。

### 3.3 Mix tau（τ）
- **目的**：ミックスドフェーズ変換時、時間領域でのプリリンギング抑制の強さを指定します。値が大きいほど早期反射の減衰が強くなり、プリリンギングが抑制されます。
- **デフォルト**：32 smp（サンプル）
- **範囲**：4 ～ 256 smp
- **推奨**：  
  - デフォルト値で適度な抑制効果が得られます。  
  - プリリンギングが気になる場合は大きく（64～128）、より自然な残響を優先する場合は小さく（8～16）設定してください。

## 4. Tail Processing（テール処理） – 新機能

IR の後半部分（テール）の高域特性を調整します。物理的な空気吸収を模擬したり、直接音・初期反射の明瞭さを保持しながらテールのみを処理することができます。

### 4.1 Tail Mode（テールモード）
- **目的**：テール処理の方式を選択します。
- **選択肢**：
  - **Air Absorption**：物理的な空気吸収を模擬し、IR 全体に高域減衰を適用します。減衰は開始周波数以上で強度に応じて連続的に強まります。
  - **Layer Tail Contouring (L1/L2)**：L0（先頭部、直接音・初期反射に相当）は無処理とし、L1/L2（後半の残響テール）にのみ減衰を適用します。直接音の明瞭さを保持しつつ、テールのざらつきを低減できます。
- **デフォルト**：Air Absorption
- **推奨**（IR長別）：

| IR長 | 推奨モード | 開始周波数 (Hz) | 強度 | パーティション強度 (※1) | 備考 |
|------|------------|-----------------|------|--------------------------|------|
| **0.5秒** | **Layer Tail Contouring** | 2000 | 0.5 | 1.0 | 直接音・初期反射を完全に保持。最も原音に忠実。 |
| | Air Absorption | 3500 | 0.3 | — | 明瞭さを保ちつつ全体に緩やかな減衰をかけたい場合。 |
| **0.6秒** | Layer Tail Contouring | 2000 | 0.5 | 1.0 | 同上。 |
| | Air Absorption | 3000 | 0.4 | — | 全体減衰でも明瞭さは保たれやすい。 |
| **0.7秒** | Layer Tail Contouring | 2000 | 0.5 | 1.0 | 同上。 |
| | Air Absorption | 2500 | 0.5 | — | デフォルト設定で自然な仕上がりに。 |
| **0.8秒** | Layer Tail Contouring | 2000 | 0.5 | 1.0 | 同上。 |
| | Air Absorption | 2000 | 0.5 | — | デフォルト設定で十分自然。 |
| **0.9秒** | Layer Tail Contouring | 2000 | 0.5 | 1.0 | 同上。 |
| | Air Absorption | 2000 | 0.5 | — | 同上。 |
| **1.0秒以上** | Air Absorption | 2000 | 0.5～0.7 | — | テールが長く、自然な減衰が適する。強度は好みで調整。 |
| | Layer Tail Contouring | 2000 | 0.5～0.7 | 1.0 | 明瞭さをより重視する場合。 |

※1: パーティション強度は Layer Tail Contouring モードのみ有効

### 4.2 Rolloff Start（減衰開始周波数）
- **目的**：減衰が始まる周波数（Hz）を設定します。この周波数以下の帯域は減衰されません。
- **デフォルト**：3500 Hz（Air Absorption）、2000 Hz（Layer Tail Contouring）
- **範囲**：20 ～ 20000 Hz
- **推奨**：  
  - 0.5秒 IR で Air Absorption を使う場合は 3500 Hz 以上に設定すると明瞭さが保たれます。  
  - 長い IR（1.0秒以上）では 2000 Hz で自然な減衰感が得られます。

### 4.3 Strength（減衰強度）
- **目的**：開始周波数以上の帯域における減衰の強さを指定します。値が大きいほど高域が強く減衰します。
- **デフォルト**：0.3（Air Absorption）、0.5（Layer Tail Contouring）
- **範囲**：0.0 ～ 2.0
- **推奨**：  
  - 0.5秒 IR で Air Absorption を使う場合は 0.3 程度に抑えると明瞭さが保たれます。  
  - 長い IR（1.0秒以上）では 0.5～0.7 が自然です。  
  - 0.0 に設定すると減衰は適用されません（旧プリセット互換）。

### 4.4 Partition Strength（パーティション強度） – Layer Tail Contouring モードのみ
- **目的**：L1/L2 レイヤー（後半テール）に適用される減衰強度の倍率を設定します。最終的な強度は `Strength × Partition Strength` となります。
- **デフォルト**：1.0
- **範囲**：0.0 ～ 2.0
- **推奨**：  
  - デフォルトで十分な効果が得られます。  
  - テールの減衰をより強めたい場合は 1.2～1.5、弱めたい場合は 0.5～0.8 に設定してください。

---

## 5. HCF / LCF / EQ LPF との関係
- **HCF（High-Cut Filter）**、**LCF（Low-Cut Filter）**、**EQ LPF（EQ ローパスフィルター）** は、リアルタイム処理（Audio Thread）で出力信号に適用されるフィルターです。  
- テール処理（Air Absorption / Layer Tail Contouring）は IR データに「焼き込まれる」ため、これらとは独立して動作します。  
- 全体の高域減衰を調整する場合は、まずテール処理で大まかな特性を決め、必要に応じて HCF や EQ LPF で微調整してください。
