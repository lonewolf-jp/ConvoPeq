# Audio Settings 画面マニュアル

Audio Settings 画面では、オーディオデバイス、信号処理の精度（ディザ／ノイズシェーパー）、オーバーサンプリング、入出力レベル調整などを設定します。

> **注意**：本アプリケーションは MIDI 機能をサポートしていません。MIDI デバイスの入出力、MIDI クロック、MIDI 学習などの機能は使用できません。

---

## 1. デバイス設定

| 項目 | 説明 |
|------|------|
| **オーディオデバイス** | 使用するオーディオドライバ／インターフェース（ASIO、WASAPI など）を選択します。 |
| **入力／出力チャンネル** | 有効にする入力・出力チャンネル数（1～2）を設定します。 |
| **サンプルレート** | 処理の基準周波数（例：44.1 kHz、48 kHz）を指定します。高いほど高域の再現性が向上しますが、CPU負荷も増加します。 |
| **バッファサイズ** | オーディオバッファの大きさ（サンプル数）。小さくするとレイテンシー（遅延）は減りますが、処理落ち（ドロップアウト）のリスクが高まります。安定しない場合は値を大きくしてください。 |

---

## 2. オーバーサンプリングフィルター種別（IIR / FIR タブ）

画面上部のタブで、オーバーサンプリング処理時のフィルター特性を選択できます。選択した内容は「オーバーサンプリング」項目の「タイプ」に反映されます。

| タブ | 名称 | 説明 |
|------|------|------|
| **IIR** | IIR (Low Latency) | 低遅延タイプ。位相に若干の影響がありますが、FIR と比較して計算負荷が低く、レイテンシーを最小限に抑えられます。 |
| **FIR** | Linear Phase (FIR) | 線形位相タイプ。位相歪みが極めて少なく、波形の整合性が求められる処理に適していますが、IIR と比較して遅延が大きくなります。 |

---

## 3. ディザ／ノイズシェーパー

| 項目 | 説明 |
|------|------|
| **ビット深度** | 出力時のターゲットビット深度（16 / 24 / 32 / Off）を設定します。高いほどダイナミックレンジを保持できます。通常は OS の出力設定（多くは 24bit または 32bit）に合わせます。「Off」では小音量信号に量子化歪みが発生する可能性があります。 |
| **ノイズシェーパータイプ** | 量子化ノイズの整形アルゴリズムを選択します。<br>- **4th-order**：4次エラーフィードバック型。軽量。<br>- **12th-order**：12次、聴覚マスキングを考慮した高品位な整形。<br>- **15th-order**：15次、より強力な整形。<br>- **Adaptive 9th-order**：9次格子フィルタ＋CMA-ESによるリアルタイム学習。 |
| **Adaptive learning… ボタン** | 「Adaptive 9th-order」選択時に有効になります。クリックすると学習専用ウィンドウが開き、現在の再生信号に合わせて最適な係数を自動探索できます。 |

---

## 4. オーバーサンプリング

| 項目 | 説明 |
|------|------|
| **タイプ** | 「オーバーサンプリングフィルター種別」タブで選択された内容（IIR または Linear Phase）が表示されます。 |
| **倍率** | Auto / 1x / 2x / 4x / 8x から選択。**Auto は現在のサンプルレートで許可される最大倍率**を自動選択します（96kHz以下で8x、192kHz以下で4x、384kHz以下で2x、それ以上で1x）。<br>**表示される倍率の上限**：現在のサンプルレートに応じて、コンボボックスに表示される倍率が変わります。<br>- 96kHz 以下：8x まで表示<br>- 192kHz 以下：4x まで表示<br>- 384kHz 以下：2x まで表示<br>- 384kHz 超：1x のみ表示<br>これらの上限を超える倍率はコンボボックスに表示されないため、選択できません。 |

---

## 5. ゲインステージング

| 項目 | 説明 |
|------|------|
| **入力ヘッドルーム** | -12.0 ～ 0.0 dB。処理前に入力レベルを減衰させ、内部オーバーフロー（クリッピング）を防ぎます。現在の処理モードに応じて上限が自動調整されます。 |
| **出力メイクアップ** | -6.0 ～ 12.0 dB。処理によるレベル損失を補うために最終出力を増幅します。こちらもモードにより範囲が変わります。 |

> **補足**：読み込まれた IR（インパルス応答）ファイルは、オーバーフローを防ぐため内部的に **-6dB** のゲインが自動適用されます。そのため、入力ヘッドルームや出力メイクアップを 0dB に設定していても、IR 自体のピークレベルは -6dB に調整されています。この自動減衰は IR のエネルギー正規化の一環であり、ユーザーが意識する必要はありません。

---

## 6. 活用のヒント

- **レイテンシーと安定性**  
  音切れやノイズが発生する場合は **バッファサイズ** を大きくしてください。リアルタイムモニタリングが必要な場合は、CPU負荷が許す限りバッファサイズを小さくします。

- **フィルター種別（IIR / FIR）**  
  **IIR** はライブ演奏など低遅延が求められる場面に適しています。**FIR** はミキシングやマスタリングなど、音質や位相の正確性を重視する場合に適しています。

- **オーバーサンプリング**  
  「Auto」は最大倍率を選択し、最も高いエイリアシング低減効果を得られます。手動で倍率を下げるとCPU負荷とレイテンシーを減らせますが、高域の品質は低下します。特に非線形処理（サチュレーションなど）では高い倍率ほど効果的です。

- **ディザリング**  
  ビット深度は出力先のデバイスに合わせてください。一般には 24bit または 32bit を選択すれば安全です。「Off」はデバッグや計測時以外は推奨しません。

- **ゲインステージングの自動連動**  
  入力ヘッドルームと出力メイクアップの許容範囲は、選択されている「処理モード」（Conv / Peq / Conv→Peq / Peq→Conv）に応じて自動的に変化します。UI上に現在のモードが表示されるため、それに従って設定してください。

- **Adaptive 9th-order 学習**  
  学習中に他のノイズシェーパータイプを選択すると、学習は中断されます。学習結果は保存され、次回「Adaptive 9th-order」を選択した際に再開できます。

以上が Audio Settings 画面の主要機能です。
