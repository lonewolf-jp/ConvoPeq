# ConvoPeq メインウィンドウ ユーザーガイド

ConvoPeq のメインウィンドウは、以下の主要セクションで構成されています。

1. **上部コントロールバー** – プリセットの保存/読み込み、処理モードの切り替え、ソフトクリップ、サチュレーション、システム状態の表示。
2. **コンボルバーコントロールパネル** – ルーム補正用FIRフィルター等の読み込み、ミックス調整、位相モード、波形表示、出力フィルター。
3. **EQコントロールパネル** – 20バンドパラメトリックEQの各バンド設定、総合ゲイン、AGC、フィルター構造。
4. **スペクトラムアナライザー** – リアルタイム周波数分析、EQ応答曲線の重ね描き、入出力レベルメーター。

以下、各セクションの詳細を説明します。

---

## 1. 上部コントロールバー

| 項目 | 説明 |
|------|------|
| **Save / Load** | 現在のすべての設定（EQ, コンボルバー, デバイス設定等）をXMLファイルに保存／読み込みます。 |
| **処理モード** | ドロップダウンで4つの信号経路を選択できます。<br>- **Conv** : コンボルバーのみ（EQはバイパス）<br>- **Peq** : EQのみ（コンボルバーはバイパス）<br>- **Conv->Peq** : コンボルバー → EQ の順に処理<br>- **Peq->Conv** : EQ → コンボルバー の順に処理<br>※ 順序を変えると音色やレイテンシーが変わります。 |
| **Soft Clip** | 出力段のソフトクリッパーを有効/無効にします。有効にすると、0dBFSを超えるピークを滑らかに丸め、デジタル・クリッピングを防ぎます。 |
| **Sat:** | ソフトクリッパーに加える非対称歪み（サチュレーション）の量を設定します（0.0～1.0）。数値ラベルをクリックすると直接編集できます。値が大きいほど真空管的な温かみが増します。 |
| **CPU** | オーディオ処理スレッドが動作している CPU コアの占有率（%）を表示します。このアプリケーションではオーディオ処理は 1 つのコアに固定されており、GUI 処理や学習機構などの他のタスクは別のコアで実行されます。そのため、表示される数値はオーディオ処理専用コアの使用率であり、他のタスクによる負荷は含まれません。 |
| **Lat** | システム全体の総レイテンシー（ミリ秒）とサンプル数を表示します。オーバーサンプリングやコンボルバーの遅延が含まれます。コンボルバー単体の遅延（Lat A / Lat T）とは異なり、前後の処理も含めた実効的な遅延です。 |
| **Audio Settings** | オーディオデバイス（入力/出力、サンプルレート、バッファサイズ、ディザ設定など）の設定画面を開きます。 |
| **?** | バージョン情報を表示します。 |

## 2. コンボルバーコントロールパネル

### 2.1 ルーム補正用FIRフィルター等の読み込みと表示
- **Load IR…** ボタン：WAV、AIFF、FLAC形式のルーム補正用FIRフィルター等を選択します。ステレオIR（2チャンネル）とモノラルIR（1チャンネル）の両方に対応しており、モノラルIRは自動的に左右同じ信号に拡張されます。読み込み後、波形が表示され、自動的にIR長が検出されます。
- **IR Advanced…** ボタン：詳細設定ウィンドウを開きます（後述）。
- **Conv Settings…** ボタン：コンボルバーのキャッシュ/アップグレード設定ウィンドウを開きます。ターゲットFFTサイズの選択、プログレッシブアップグレードの有効/無効、IRキャッシュエントリ数の調整、キャッシュクリアが行えます。

#### IR波形グラフ
読み込まれたIRの時間波形（振幅の絶対値のピーク）を表示します。横軸は時間（ミリ秒または秒）、縦軸は正規化された振幅です。波形の減衰の様子を視覚的に確認できます。

#### IR情報表示（IR情報ラベル）
- **IR名**：読み込まれたファイル名（拡張子なし）。
- **Auto**：自動検出されたIR長（秒）が表示されます。手動で「IR Length」を変更した場合は `[Manual]` と表示され、自動検出値も括弧内に参考表示されます。
- **Lat A**：コンボルバーのアルゴリズムが持つ固定遅延（サンプル換算のミリ秒）。主にパーティションサイズに依存します。
- **Lat T**：アルゴリズム遅延（Lat A）に加え、IRのピーク位置に起因する追加遅延（位相処理による影響）を含めた**合計遅延**。実際に出力信号がどの程度遅れるかの目安です。
- **上部コントロールバーの「Lat」との違い**：上記はコンボルバー単体の遅延ですが、上部の「Lat」はオーバーサンプリングやEQ、出力フィルターなど、システム全体の遅延を**ベースサンプルレート換算**で表示したものです。コンボルバー単体の遅延だけでなく、前後の処理も含めた実効的なレイテンシーを確認したい場合は上部の「Lat」をご覧ください。

### 2.2 基本コントロール
- **Mix** スライダー：ドライ（原音）とウェット（畳み込み後）のバランスを調整します（0% = 原音のみ、100% = 畳み込みのみ）。
- **Smoothing** スライダー：ドライ/ウェットのミックス値が変更されたとき、その変化を滑らかにする時間（ミリ秒）を設定します。急激な切り替えによるクリックノイズや音量の急変を防ぎます。値が大きいほど変化は緩やかになります。
- **位相モード** ドロップダウン：
  - **As-Is**：元のIRの位相を保持します。
  - **Mixed**：低域では線形位相（定位感や位相歪みの少なさ）の利点を活かし、高域では最小位相（プリリンギングの抑制）の利点を活かすことで、両者のバランスを取ったモードです。特に音楽再生や映画鑑賞など幅広い用途で、自然な音場感と過渡応答のクリアさを両立しやすいため、標準的な設定として推奨しています。遷移周波数は詳細設定で調整可能です。
  - **Minimum**：最小位相に変換します。プリリンギングが減少しますが、位相特性が変化します。
- **Resample Phase**：IR のサンプルレートがプロジェクトと異なる場合のリサンプリング位相モードを選択します。
  - **Linear (precise)**：周波数特性を正確に保持（測定・ミキシング用途向け）。
  - **Minimum (musical)**：アタック成分を保持（ギターキャビネット等の音楽用途向け）。
  変更すると IR の再構築がトリガーされます。
- **Exp Direct Head**（実験的機能）：有効にすると、IRの先頭部分（直接音）を畳み込みエンジンの遅延を経由せず、ゼロ遅延で出力に加えます。この設定を変更すると IR の再構築（リビルド）がトリガーされます。**通常のWet/Dryミックス機能はそのまま使用できます**が、直接音が早く到達するため時間的なバランスが変化します。この機能は、ライブ演奏時のモニタリングなど、極力遅延を減らしたい場合に検討してください。なお、実験的機能のため、IR全体の周波数特性や位相特性に影響を与える可能性があります。

### 2.3 出力周波数フィルター（コンボルバーが最終段の場合のみ有効）
- **HCF**（ハイカットフィルター）：  
  - **Sharp**：急峻なロールオフ（Butterworth 4次 Q1=0.5412, Q2=1.3066）
  - **Natural**：Linkwitz-Riley 4次 Q=0.7071（位相特性良好、デフォルト）
  - **Soft**：穏やかなロールオフ（2次 Q=0.5）
  カットオフ周波数はサンプルレートに応じて自動設定されます（48kHz以下で19kHz、それ以上で22kHz）。
- **LCF**（ローカットフィルター）：  
  - **Natural**：18Hz Butterworth 2次（デフォルト）
  - **Soft**：15Hz Q=0.5 の2次HPF
  いずれもDC成分と超低域ノイズを除去します。

### 2.4 詳細設定ウィンドウ（IR Advanced…）
- **IR Length**：IRの実効長を手動で設定（0.5～3.0秒）。自動検出長（Auto）も表示。
- **Rebuild Debounce**：パラメータ変更後の再構築待機時間（ms）。デフォルト20ms、範囲10～3000ms。
- **Mixed Phase Parameters**：
  - **Mix Start f (f1)**：As-Is から Mixed への遷移開始周波数（100～400 Hz、デフォルト200 Hz）。
  - **Mix End f (f2)**：Mixed から Minimum への遷移終了周波数（700～1300 Hz、デフォルト1000 Hz）。
- **Tail Processing**：
  - **Tail Mode**（Air Absorption / Layer Tail Contouring / Bypass）。
  - **Tail Start**：テール処理開始位置（0.01～0.80 秒、デフォルト0.085 秒）。
  - **Tail Strength**：減衰強度（0.0～2.0、デフォルト1.0）。
  - **Tail L1-L2 Multiplier**：Layer Tail Contouring モード時の L1/L2 倍率（2～16、デフォルト8）。
- **Resample Phase**：IR リサンプリング時の位相モード（Linear precise / Minimum musical）。

## 3. EQコントロールパネル

### 3.1 バンド別コントロール
- 20バンドのパラメトリックEQです。各バンドに以下の操作があります：
  - **ON/OFF**：バンドの有効/無効を切り替えます。
  - **タイプ**（Low Shelf / Peaking / High Shelf / Low Pass / High Pass）
  - **チャンネル**（Stereo / Left / Right / Mid / Side）：フィルタを適用するチャンネルを選択。Mid/Side は M/S 処理に対応し、各バンドごとに独立したチャンネルモードを設定できます。
  - **ゲイン**（dB） – 数値入力で設定。
  - **周波数**（Hz） – 数値入力で設定。
  - **Q値** – 数値入力で設定。

### 3.2 全体コントロール（右上）
- **Total Gain**：EQ全体の出力ゲイン（dB）。AGC有効時は無効化されます。
- **AGC**：自動ゲインコントロール。入出力レベルの差を補正し、全体の音量を安定化します。
- **Sat:**：SVF（State Variable Filter：状態変数フィルター）の非線形飽和量（0.0～1.0）。数値ラベルをクリックすると直接編集できます。SVFはフィルターの内部状態に非線形性を加えることで、真空管のような倍音成分を生成します。
- **Structure**：フィルターの接続方式。
  - **Serial**：各バンドを直列接続（シンプル・低負荷）。
  - **Parallel**：並列接続。位相特性が変化し、より自然な残響感を得られる場合があります。（高負荷）
- **Reset**：すべてのバンドをデフォルト設定（ゲイン0dB、Q=0.707、タイプはBand1/20がシェルフ、他はピーキング）に戻します。

### 3.3 EQ出力フィルター（EQが最終段の場合のみ有効）
- **HCF**（ハイカットフィルター）：  
  - **Sharp**：急峻なロールオフ（Q=1.0 の2次×2段）  
  - **Natural**：Linkwitz-Riley Q=0.7071 の2次×2段（位相特性良好、デフォルト）  
  - **Soft**：穏やかなロールオフ（Q=0.5 の2次×2段）  
  カットオフ周波数はサンプルレートに応じて自動設定されます（48kHz以下で19kHz、それ以上で24kHz）。なお、このフィルターには20Hz Butterworth 2次のハイパスフィルターが常に直列に挿入され、スピーカー保護と超低域ノイズ除去を行います。

## 4. スペクトラムアナライザー

### 4.1 表示エリア
- **周波数軸**：対数スケールで 20Hz ～ 20kHz を表示。低域をやや拡大した特殊マッピングにより見やすくなっています。
- **レベル軸**：-80dB ～ +20dB の範囲。グリッド線は 20dB ごと、0dB 線は赤で強調表示されます。
- **スペクトラム**：リアルタイムFFT（4096点、75%オーバーラップ）による周波数分析結果。棒グラフの色はレベルに応じて青→水色→黄→赤に変化します。
- **ピークホールド**：白い水平線でピークレベルを保持し、1秒後に徐々に減衰します。
- **EQ応答曲線**：現在のEQ設定による総合周波数特性を白（左チャンネル）と赤（右チャンネル）の折れ線で重ねて表示します。個別バンドの応答も L/R/Mid/Side の4チャンネルごとに薄い色で表示されます。
- **周波数ポイント**：各バンドの現在の周波数位置に丸印が表示され、EQの設定状態が一目でわかります。

### 4.2 右側レベルメーター
- **IN / OUT**：入力と出力の瞬時レベル（dB）を縦棒で表示。0dB基準線は赤、ピーク値は白い針で保持されます。
- **ボタン**：左下に「Analyzer: Input / Analyzer: Output」の切り替えボタンがあり、スペクトラムの分析対象を入力信号か出力信号か選択できます。
  右隣の「Analyzer: ON / Analyzer: OFF」でアナライザーの表示をオン/オフできます（オフ時はCPU負荷軽減）。

## 5. プリセットの保存と読み込み
- **Save**：現在のすべての設定（EQ、コンボルバー、デバイス設定、テール処理パラメータ等）をXMLファイルに保存します。
- **Load**：保存したXMLファイルを読み込み、状態を復元します。また、以下の形式のテキストファイルも読み込めます（その場合、EQ設定のみが復元されます）。
  - **Equalizer APO 設定ファイル**（.txt）：複雑な設定（複数バンドの組み合わせ等）には完全には対応していませんが、基本的なパラメトリックEQ設定を読み込むことができます。
  - **AutoEq ヘッドフォン設定ファイル**：AutoEqの「Custom Parametric Eq」形式でエクスポートしたファイルに対応しています。ヘッドフォン補正用のEQカーブを簡単に適用できます。

---

## 6. ヒントと注意事項
- **IR長**：0.5秒以下は指定できません。室内補正用IRは0.5～1.0秒が一般的です。
- **テール処理**：IR長が短い（0.5秒付近）場合は、**Layer Tail Contouring** モードが直接音の明瞭さを保つのに適しています。
- **AGC**：有効にするとTotal Gainが自動調整されるため、手動のTotal Gainは無効になります。
- **処理モードの順序**：コンボルバーを先に配置（Conv→Peq）すると、EQが残響の特性をさらに調整できます。Peq→Conv は、EQで音色を整えた後に空間特性を付加するのに適しています。
- **レイテンシー**：コンボルバーのレイテンシーは、IR長とオーバーサンプリング設定に依存します。現在の総レイテンシーは上部バーに表示されます。
